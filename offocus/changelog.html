Openfire Focus Provider Plugin Changelog

0.9.3 -- November 3, 2017

  • Now requires Java 8 to run.
  • Wait for OFMeet to have started before loading OFFocus.

0.9.2 -- August 15, 2017

  • Fixed Issue #12: Ensure that the 'focus' user is a sysadmin of the conference service(s).
  • Fixed Issue #14: Should not imply that focus user logs in anonymously.
  • Updated Jitsi Jicofo from 1.0-20170523.202534-1 to 1.0-20170809.150330-7.

0.9.1 -- June 26, 2017

  • Reduced minimum Openfire version dependency from 4.2.0 to 4.1.5.
  • Fixed Issue #4: OFFocus plugin fails to initialize with CME exception.
  • Fixed Issue #5: Fails to provision 'focus' user.

0.9.0 -- June 1, 2017

  • Initial release, based on code that was previously in Openfire Meetings.
  • Depends on Jicofo version 1.0-20170523.202534-1
