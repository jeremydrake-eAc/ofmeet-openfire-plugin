Openfire Meetings Plugin Changelog

0.9.3 -- November 3, 2017

  • Now requires Java 8 to run.
  • Fix the change in meeting URLs from version 0.3.x to 0.9.x
  • Finishing the Quartz CRON job triggers for meetings.
  • Add support for notifications via IM as used by Pade.
  • Fix conference addressing in admin console.
  • Improve parameter encoding in admin console.
  • Adds missing closing slash in websocket URL.
  • Uses websocket when using Openfire 4.2.0 or later.

0.9.2 -- August 15, 2017

  • Updated Jitsi Meet from 1.0.2049 to 1.0.2207.
  • Updated Jitsi's Openfire plugin for Videobridge to version 2.1-20170815.124603-3 (which depends on Videobridge version 1.0-20170811.042244-386).
  • Added Support for vertical filmstrip.

0.9.2 -- August 15, 2017

  • Removed Global Intercom support, which was a deprecated, proprietary feature.
  • Implemented Web application path should be configurable (and default to /ofmeet instead of /ofmeet/jitsi-meet).
  • Updated Jitsi Meet from 1.0.2049 to 1.0.2207.
  • Updated Jitsi's Openfire plugin for Videobridge to version 2.1-20170815.124603-3 (which depends on Videobridge version 1.0-20170811.042244-386).
  • Added Support for vertical filmstrip.

0.9.1 -- June 26, 2017

  • Reduced minimum Openfire version dependency from 4.2.0 to 4.1.5.
  • Fixed Issue #5: Fails to provision 'focus' user.
  • Depends on Jitsi's Openfire plugin for Videobridge version 2.1-20170626.093604-1 (which depends on Videobridge version 1.0-20170623.182232-366).
  • Updated Jitsi Meet from 1.0.1992 to 1.0.2049.

0.9.0 -- June 1, 2017

  • Re-release after many structural changes, previously known as 'the 0.4.x branch'.
  • No longer contains Jicofo (please combine this plugin with the new offocus plugin)
  • Depends on Jitsi's Openfire plugin for Videobridge version 2.0-20170601.084101-8 (which depends on Videobridge version 1.0-20170531.151022-361).
  • Depends on Jitsi's Meet web application version 1.0-1992.
